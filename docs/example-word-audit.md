# JamoFlow Example Word Audit

기준일: 2026-03-24

## 목적

현재 `src/data/jamoData.ts`에 들어간 예시 단어를 [example-word-selection-criteria.md](./example-word-selection-criteria.md) 기준으로 검수하고, 교체 우선순위를 정리한다.

## 현재 상태

- 데이터 구조는 구현 완료
- 자모 상세 화면에 예시 단어 노출 완료
- 1차 단어 품질 검수와 교체 반영 완료

## 검수 기준

1. 입문 학습자에게 친숙한가
2. target jamo가 글자 안에서 분명한가
3. 뜻풀이가 간단한가
4. 다른 자모 학습용으로 더 적합한 단어를 끌어온 것은 아닌가
5. 지나치게 다의적이거나 문맥 의존적이지 않은가

## 초기 관찰

- 상당수 단어는 MVP용으로 무난하다.
- 일부 단어는 뜻이 여러 개이거나 학습 시작 단어로는 덜 직관적일 수 있다.
- 자음보다 모음 쪽에서 단어의 직관성이 더 중요하다.

## 우선 검토 후보

## 1차 교체 반영

- `다리` -> `달`
  - 다의성을 줄이고 더 직관적인 초급 명사로 교체
- `차` -> `치약`
  - `car/tea` 다의성을 제거
- `이` -> `이불`
  - 극단적 다의성을 제거
- `호수` -> `하마`
  - 더 쉬운 concrete noun으로 교체
- `크다` -> `그네`
  - 서술어 예시를 초급 명사로 교체
- `리본` -> `모래`
  - 차용어보다 더 기본적인 everyday noun으로 교체
- `주스` -> `집`
  - 차용어 의존도를 줄임
- `쿠키` -> `칼`
  - 더 기본적인 core noun으로 교체
- `타조` -> `태양`
  - 2음절 concrete noun으로 조정해 기준 적합성을 높임
